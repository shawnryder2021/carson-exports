How to Embed on Your Website

Add this code to the <head> section of your HTML:

<script>
window.DealerAIConfig = {
  serverUrl: "YOUR_SERVER_URL",
  dealerName: "Your Dealership Name",
  primaryColor: "#1e6fff",
  position: "bottom-right",
  theme: "auto"
};
</script>
<script src="YOUR_SERVER_URL/chat-widget.js" defer></script>

Self-Contained Widget

The chat widget loads all required assets (fonts, icons, styles) automatically. No additional setup needed.

Session Persistence

Conversations persist across page navigation using sessionStorage, so users don't lose their chat history.

Page-Aware AI

The AI reads page context (title, meta, vehicle info) to provide relevant, contextual responses.

Fully Customizable

Configure colors, position, theme, and dealer name through the DealerAIConfig object.

JavaScript API

Control the widget programmatically: DealerAI.open(), DealerAI.close(), DealerAI.sendMessage()

Mobile Responsive

The chat window goes full-screen on mobile devices for the best user experience.
